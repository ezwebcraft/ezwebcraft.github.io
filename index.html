➔ Docker Update Images

Published: Wednesday, Nov 27, 2019   |   1 minute read
A great way to update docker images is to use this one-liner. Since, docker does not have a command; this command will update images that you have already pulled. The only way is to pull all images again using docker pull command. This simple one-liner can help you update all images at once. docker images |grep -v REPOSITORY|awk '{print $1}'|xargs -L1 docker pull
